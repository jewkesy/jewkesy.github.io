About Me

I have an extensive background in software development, creating content-rich web applications and background services using established technologies and design patterns.

Employed as an innovation developer working with new and up-and-coming technologies to create innovative concepts and prototypes for future markets and opportunities.

I'm currently working with Node.JS and MongoDB, this is evident if you take a look at my repos. I'm loving the performance and scalability of the MEAN stack and the friendly, pro-active community support that comes with it - a refreshing change from my days working on .NET solutions.

You can also find me on the education discussion boards at MongoDB, which I'm a heavy contributor to since starting a series of online training certifications.

When not attached to a keyboard, you'll find me on my mountain bike exploring woodland trails and trying to get as muddy as possible.

My Projects

A brief overview of some of my projects that I'm currently working on: -

FlappyJS

Jumping on the FlappyBird bandwagon, a complete browser-based game written in #HTML5. Based upon the classic original and with graphical help from a few versions that exist already.

=== Play Now! ===

More of a playground for learning about the HTML5 canvas object in modern browsers. My focus is understanding features like touch/event-driven behaviour across multiple devices and browsers, and game-specific design patterns for collision detection and sprite animations.

It is great fun to play so I'm updating this game often with new features and fixing any bugs found. The wife is addicted to it and puts in the change requests...

FormulaOneStats

AngularJS application which focuses on the MVW design pattern to consume a public Formula One REST API.

Using the great UI Router and Bootstrap for fluid, cross-device responsiveness while learning the core features of AngularJS.

I'm slowly adding components (there is over 60 years of stats!) then pushing them to the master branch as and when I can. Hopefully all done in time for the new season

Nodopoly

What do you get when you cross Node.JS, MongoDB and Monopoly? Nodopoly!

Self-made challenge to get a better understanding of these technologies and new features to create an online game that I'd probably play.

Still early days, a bit of ground work to do before anything appears remotely resembling the game in a browser window. Public-facing branch is in my repos.

KeyStageWords

JQuery web application to help teach children Key Stage and High Frequency words. Works on any device. Works on my machine. Tested by my daughter and her school, now used by many. Includes free kitten.

TeachTheTime

A work-in-progress JQuery web application to compliment the KeyStageWords application, this project is designed to help teach children how to learn the time.

SocialMe (closed beta)

An Android application that uses a wide variety of social networking feeds, such as Twitter and Instagram, that allows users to discover nearby events, follow friends and discover new followers.
